# Heard_Island_Synthetic_experiments
This repository contains the code used to reproduce the simulations, data analysis, and figures of  Yang, W., Shapero, D.R., Maurer, J., Tielidze, L.G., and Mackintosh, A.N. (2026): Sensitivity of Glacier Evolution on Small Islands to Internal Dynamics and Terminus Type: Insights from Heard Island.
