# Heard_Island_Synthetic_experiments
This repository contains the code used to reproduce the simulations, data analysis, and figures of  
- Yang, W., Shapero, D.R., Maurer, J., Tielidze, L.G., and Mackintosh, A.N. (2026): Sensitivity of Glacier Evolution on Small Islands to Internal Dynamics and Terminus Type: Insights from Heard Island.


We systematically compare multiple dynamical schemes and associated parameters to evaluate their influence, and assess the sensitivity of glaciers on Heard Island using a 2D finite-element model, Icepack [https://icepack.github.io] developed by Shapero and others (2021).
